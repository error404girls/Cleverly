Cleverly

Cleverly este o aplicație web interactivă destinată celor care își doresc să devină mai organizați, să învețe mai eficient și să își gestioneze mai bine timpul. Aceasta combină funcții esențiale de productivitate și sprijin pentru concentrare cu instrumente simple și practice pentru reținerea informației.

1. Descriere
Aplicația oferă un spațiu centralizat unde utilizatorii pot:
- crea liste de sarcini,
- genera la quiz-uri rapide,
- genera fișe sintetizate de învățare,
- folosi timerul de tip Pomodoro pentru concentrare.

Totul este accesibil direct din browser, fără instalare sau cont.

2. Funcționalități
•	Task manager (to-do list) – pentru organizarea activităților zilnice.
•	Generator de quiz-uri interactive – pentru testarea rapidă a cunoștințelor.
•	Generator de flashcarduri (fișe de învățare) – generare automată de rezumate pentru o reținere mai bună.
•	Timer de tip Pomodoro – pentru îmbunătățirea atenției și evitarea distragerilor.
•	Interfață responsive – optimizată pentru utilizare pe mobil, tabletă și desktop.

3. Tehnologii folosite
- HTML5
- CSS3
- JavaScript (vanilla)
